# machi_coro
Machi Koro (Board Game)
Цифровая версия настольной игры Machi Koro.

Данный проект предпологает игру только на 4 игрока и только локально

## Запуск проекта локально

### 1. Клонирование репозитория

git clone https://github.com/g1mberg/machi_coro

### 2. Запуск сервера и игры

Запустить файл run_test.bat и у вас откроется сервер и 4 окна с игрой
### можно играть

### Как играть 

Запустите игру

По очереди:

  бросайте кубик

  получайте доход

  покупайте здания

Побеждает игрок, который первым построит все достопримечательности
### Ознакомиться с  полными правилами игры можно по ссылке https://hobbygames.ru/download/rules/Machi_Koro_rules-web_2015.pdf
